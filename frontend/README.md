# Task Manager

Frontend of app "Task Manager"

### Stack

- Main:
    - Angular
- UI Library:
    - NgZorro

### Commands

Run in development mode:
```sh
cd frontend
npm i
npm start
```

Run build:
```sh
cd frontend
npm i
npm build
```

Run linter:
```sh
cd frontend
npm i
npm run lint
```

Run tests:
```sh
cd frontend
npm i
npm test
```
