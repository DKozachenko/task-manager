# Task Manager

Backend of app "Task Manager"

### Stack

- Main:
    - Node JS
    - Express
    - MongoDb
- Authentication:
    - Passport
    - Passport-jwt
- Password hashing:
    - Bcryptjs
- Logging:
    - Morgan
    - Winston
- Testing
    - Jest
    - Supertest

### Commands

Run in development mode:
```sh
cd backend
npm i
npm run dev
```

Run in production mode:
```sh
cd backend
npm i
npm start
```

Run linter:
```sh
cd backend
npm i
npm run lint
```

Run tests:
```sh
cd backend
npm i
npm test
```
